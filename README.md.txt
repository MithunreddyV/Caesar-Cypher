# Caesar Cipher using wxPython

## Description
This project implements a Caesar Cipher encryption and decryption tool
using Python and wxPython GUI framework.

## Features
- Encrypt and decrypt text using Caesar Cipher
- Simple and user-friendly GUI
- Supports both uppercase and lowercase letters

## Technology Used
- Python
- wxPython

## How to Run
1. Install wxPython:
   pip install wxPython
2. Run the program:
   python Caesar Cipher
